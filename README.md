1. **Enter URL**: Input any valid URL in the target field
2. **Customize Appearance**: 
   - Adjust colors using color pickers
   - Upload optional logo (PNG/JPG, max 2MB)
3. **Advanced Settings**:
   - Set error correction level
   - Adjust size and margins
4. **Export**: Download as PNG, SVG, or PDF
